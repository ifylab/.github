# Code of Conduct

The `.ify` project adopts the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/), version 2.1, as its code of conduct.

In short: be respectful and constructive. Treat others well.

To report a concern, contact **goldsmith323@gmail.com**. Reports are handled confidentially.

This file lives in the org `.github` repository and applies to every `.ify` repository that does not provide its own.
